8-Track
I consider myself somewhat of an audiophile. I listen to music while I work, when I'm driving, cooking, or working out, and I find myself listening to hundreds of new albums every year, including lots of recommendations from friends and family. But I didn't have a good way to keep track of all these albums other than in a notes app on my phone. This is how 8-Track was born. I wanted a simple way to search artists and albums and add them to a concise list of recommendations without sacrificing aesthetic. 8-Track is built using the Django framework for Python with a simple front-end made with vanilla JavaScript. It queries the Spotify API with the help of the Python wrapper "Spotipy" to return album and artist data that the user can add to their own personalized "Rec List." 8-Track was my final project at Momentum Learning, and was made with two other developers.

Snip-It!

Snip-It! is an app I developed as another of my projects during Momentum's Full Stack Development course. It allows users to catalogue and share commonly used code snippets with other developers. Each snippet can be copied to the clipboard with the click of a button, and users can search the Snip-It! library by language, keywords, or other snippets of code. It is built in the Django framework for Python with a vanilla JavaScript front-end. The syntax highlighting is supported by Prism.js and currently supports HTML, CSS, JavaScript, and Python.
